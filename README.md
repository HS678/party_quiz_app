# 入党积极分子在线刷题网页

基于题库文档生成的本地 Streamlit 刷题网页。题目编号与原文档一致，答题前不显示答案，答题后显示对错、正确答案和题目讲解。

## 功能

- 随机单选题：单选题随机打乱练习。
- 随机多选题：多选题随机打乱练习。
- 顺序刷题：按照原文题号顺序练习，先刷单选题，再刷多选题。
- 模拟考试：随机抽取单选 70 道、多选 30 道，每题 1 分，满分 100 分；前 70 题固定为单选题，后 30 题固定为多选题，并显示“第一部分 单项选择题”“第二部分 多项选择题”。
- 错题本：记录答错次数、答错选项、正确答案和题目讲解。

## 题库规模

- 单选题：306 道
- 多选题：78 道
- 总题量：384 道

## 快速运行

进入项目目录：

```bash
cd party_quiz_app
```

安装依赖：

```bash
pip install -r requirements.txt
```

启动网页：

```bash
streamlit run app.py
```

浏览器会自动打开。如果没有自动打开，手动访问：

```text
http://localhost:8501
```

## 测试

```bash
pytest
```

测试内容包括：

- 题库总数校验
- 单选/多选数量校验
- 题号连续性校验
- 答案选项合法性校验
- 模拟考试题量是否足够
- 模拟考试是否满足前 70 题单选、后 30 题多选
- 模拟考试分区标题校验
- 顺序刷题顺序校验

## 文件说明

```text
party_quiz_app/
├── app.py                         # Streamlit 网页主程序
├── data/questions.json             # 结构化题库
├── data/wrong_book.json            # 运行后自动生成的错题本
├── source/入党积极分子题库答案第一版.docx
├── scripts/build_questions.py      # 从文档解析题库的脚本
├── tests/test_question_bank.py     # 自动测试
├── requirements.txt
└── README.md
```

## 注意

错题本保存在本机项目目录下的 `data/wrong_book.json`。如果删除该文件，错题记录会清空，但题库不会受影响。

## 本机进度保存

本版本适合在自己电脑本地使用，会把学习进度保存到：

```text
data/progress.json
```

保存内容包括：

- 随机单选题：随机顺序、当前进度、是否已提交当前题
- 随机多选题：随机顺序、当前进度、是否已提交当前题
- 顺序刷题：当前进度、是否已提交当前题
- 模拟考试：本次试卷题目、已选择答案、是否已提交

关闭网页或关闭命令行后，重新运行：

```bash
streamlit run app.py
```

会自动继续上次进度。

如果想从头开始，可以在左侧栏点击：

```text
清空本机进度
```

注意：如果部署到 Streamlit Community Cloud，该进度文件会保存在云端运行环境，不适合作为多用户长期进度记录。自己电脑本地使用最合适。
